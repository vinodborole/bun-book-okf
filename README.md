# bun-book-okf

An **[OKF](https://github.com/GoogleCloudPlatform/knowledge-catalog/blob/main/okf/SPEC.md) knowledge bundle** of [https://bun.sh/docs](https://bun.sh/docs) — ready for agents to read and for you to chat with.

```bash
pip install okf-kit
okf get bun-book
okf chat bun-book --provider ollama
```

Built and kept fresh with [okf-kit](https://github.com/vinodborole/okf-kit).
